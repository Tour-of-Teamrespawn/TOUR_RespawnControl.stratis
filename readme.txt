Respawn Control

Made by Mr.Ben 

Tour of Teamrespawn 

Version 11

Implement the TOUR_RC folder into your mission. Execute the init file as shown in the example mission params.sqf.
See description.ext for code to add for the setup of the mission. 
Compatible with ACE medical to detect when all players within a side are dead or incapacitated - use TOUR_RC_WEST_DEAD for example to detect for end of mission.

Feel free to use these scripts however do not alter. If changes are needed, please request.
